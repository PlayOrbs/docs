# PORB Token

**PORB** is the native token of the PlayOrbs ecosystem.

## Token Specification

| Property | Value |
|----------|-------|
| **Name** | PORB |
| **Symbol** | PORB |
| **Decimals** | 9 |
| **Smallest Unit** | 1 atom = 10⁻⁹ PORB |
| **Total Supply Cap** | 100,000,000 PORB |
| **Mint Authority** | PDA-controlled |

## Token Utility

| Use Case | Description |
|----------|-------------|
| **Emissions Reward** | First 3 joiners per round can earn PORB |
| **Season Pool** | Proportional rewards for leaderboard performance |
| **Governance** | Future voting on protocol parameters |
| **Tournament Entry** | Special events may require PORB |
| **Cosmetics/NFTs** | Unlock exclusive in-game items |

## Fair Launch Principles

PORB follows a true fair launch model:

| Allocation | Amount | Purpose |
|------------|--------|---------|
| Team/Advisors | 0% | None |
| Investors | 0% | None |
| Treasury | 0% | None |
| **Gameplay** | **100%** | Emissions + Season Pools |

## How PORB is Earned

### 1. Per-Round Emissions
First 3 players to join a round can receive PORB:

| Position | Amount |
|----------|--------|
| 1st Joiner | 100 PORB |
| 2nd Joiner | 60 PORB |
| 3rd Joiner | 40 PORB |

### 2. Season Pool
Each emission also contributes 50 PORB to the season pool, distributed proportionally at season end.

### 3. Genesis Conversion
Season 0 players can convert points to PORB at genesis (50 points = 1 PORB, up to 250k PORB cap).

## What Makes PORB Different

### Liquidity-Backed
Every PORB token is backed by real game fees accumulated in the CLMM pool.

### Deflationary Pressure
- Buyback mechanism purchases PORB with protocol fees
- Purchased PORB adds to locked liquidity
- Creates continuous buy pressure

### Sustainable Emissions
- Probabilistic model prevents rapid depletion
- Epoch decay ensures 100+ year longevity
- Hard cap at 100M prevents inflation

### Verifiable Scarcity
- On-chain supply tracking
- Mint authority PDA-controlled
- No backdoor minting possible
