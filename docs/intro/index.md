# What is PlayOrbs?

**PlayOrbs** is a decentralized battle royale gaming platform powered by two leading blockchains: [Solana](https://solana.com) for high-performance on-chain execution and the [Internet Computer (ICP)](https://internetcomputer.org) for provably fair randomness. Players compete in tiered rounds, paying entry fees in SOL for a chance to win prize pools and earn PORB tokens.

## Value Proposition

- **Fair Launch Economics**: No pre-mine, no team allocation - tokens are earned through play
- **Real Liquidity Backing**: Every PORB token is backed by accumulated game fees
- **Dual-Chain Security**: [Solana](https://solana.com) handles settlements while [ICP](https://internetcomputer.org) provides cryptographic randomness

## Key Differentiators

| Feature | PlayOrbs | Traditional P2E |
|---------|-----------|-----------------|
| Token Distribution | Earned through gameplay | Pre-allocated to team/investors |
| Liquidity Source | Accumulated game fees | External capital injection |
| Token Utility | Prize pools, emissions, governance | Often speculative only |
| Settlement | Verifiable on-chain | Often centralized servers |

## Powered By

### [Solana](https://solana.com) — Execution Layer
Built with Anchor 0.32.1, the on-chain program manages:
- Round creation and settlement
- Prize pool distribution
- Token emissions
- Player statistics

### [Internet Computer (ICP)](https://internetcomputer.org) — Randomness Layer
ICP canisters provide threshold cryptography for:
- Unpredictable seed generation
- Verifiable fairness via Merkle proofs
- Tamper-proof settlement that no one can manipulate

### Deterministic Physics
The 120Hz game engine guarantees:
- Byte-perfect replay across platforms
- Trustless verification
- Fair competition
